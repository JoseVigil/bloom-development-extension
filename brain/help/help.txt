
Brain CLI - Modular system for Bloom (Executable)

╭──────────────────────────────────────── Uso / Usage ────────────────────────────────────────╮
│                                                                                             │
│  Brain CLI - Ejecutable standalone                                                          │
│                                                                                             │
│  Uso:                                                                                       │
│    brain [OPTIONS] <category> <command>                                                     │
│                                                                                             │
│  Ejemplos:                                                                                  │
│                                                                                             │
│    # Comando básico                                                                         │
│    brain health native-ping                                                                 │
│                                                                                             │
│    # Con flags globales                                                                     │
│    brain --json nucleus list                                                                │
│                                                                                             │
│    # Con verbose                                                                            │
│    brain --verbose profile create 'My Profile'                                              │
│                                                                                             │
│  [!] Los flags globales (--json, --verbose) van ANTES de <category>                         │
│                                                                                             │
╰─────────────────────────────────────────────────────────────────────────────────────────────╯

╭─────────────────────────────── Opciones Globales / Global Options ───────────────────────────────╮
│   --json       Salida en formato JSON (debe ir ANTES del comando)                                │
│   --verbose    Habilitar logging detallado (debe ir ANTES del comando)                           │
│   --help       Mostrar este mensaje de ayuda                                                     │
╰─────────────────────────── Estas opciones deben ir después de 'brain' ───────────────────────────╯
