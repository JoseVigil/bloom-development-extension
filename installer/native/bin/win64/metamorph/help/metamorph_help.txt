
                                                       METAMORPH
                                                System State Reconciler


------------------------------------------------------------------------------------------------------------------------
  USAGE
------------------------------------------------------------------------------------------------------------------------

  metamorph [OPTIONS] <command> [args]

  Quick examples:
    metamorph version                    # Display version information
    metamorph info                       # Show system information
    metamorph --json info              # JSON output for automation


------------------------------------------------------------------------------------------------------------------------
  GLOBAL OPTIONS
------------------------------------------------------------------------------------------------------------------------

  --json           Output in JSON format (machine-readable)
  --verbose        Enable detailed logging for debugging
  --help           Show this help message


------------------------------------------------------------------------------------------------------------------------
  COMMAND CATEGORIES
------------------------------------------------------------------------------------------------------------------------

  SYSTEM            System information and version details                   2 cmds
  INSPECTION        Binary and state inspection tools                        3 cmds
  RECONCILIATION    Manifest-driven state reconciliation                     2 cmds

  > INSPECT
    Inspect all binaries (AppData or native build output)

    Usage: metamorph inspect

    Flags:
      --all, -a                  Include external binaries (Temporal, Ollama, Chromium, Node)
      --native                   Inspect native/bin/<platform>/ build output instead of AppData

    Example:
        metamorph inspect
        metamorph inspect --all
        metamorph inspect --native
        metamorph inspect --native --all
        metamorph --json inspect
        metamorph --json inspect --native
        metamorph --json inspect --all

  > ROLLOUT
    Deploy built binaries from repo to AppData

    Usage: metamorph rollout [--only <component>] [--dry-run]

    Arguments:
      [--only               required
      <component>]          required
      --dry-run             optional

    Flags:
      --dry-run                  Preview what would be copied without making changes
      --only                     Deploy a single component instead of all (brain, nucleus, sentinel, metamorph, conductor, setup, host, cortex, nssm, bootstrap, vsix, node)

    Example:
        metamorph rollout
        metamorph rollout --dry-run
        metamorph rollout --only brain
        metamorph rollout --only nucleus
        metamorph rollout --only sentinel
        metamorph rollout --only metamorph
        metamorph rollout --only conductor
        metamorph rollout --only setup
        metamorph rollout --only host
        metamorph rollout --only cortex
        metamorph rollout --only nssm
        metamorph rollout --only bootstrap
        metamorph rollout --only vsix
        metamorph rollout --only node
        metamorph rollout --only brain --dry-run
        metamorph --json rollout --only nucleus

      {
        "status": "success",
