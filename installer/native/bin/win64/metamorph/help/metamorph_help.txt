
                                                       METAMORPH
                                                System State Reconciler


------------------------------------------------------------------------------------------------------------------------
  USAGE
------------------------------------------------------------------------------------------------------------------------

  metamorph [OPTIONS] <command> [args]

  Quick examples:
    metamorph version                    # Display version information
    metamorph info                       # Show system information
    metamorph --json info              # JSON output for automation


------------------------------------------------------------------------------------------------------------------------
  GLOBAL OPTIONS
------------------------------------------------------------------------------------------------------------------------

  --json           Output in JSON format (machine-readable)
  --verbose        Enable detailed logging for debugging
  --help           Show this help message

  COMMAND CATEGORIES

=====================================================[ INSPECTION ]=====================================================

  > INSPECT
    Inspect all binaries (AppData or native build output)

    Usage: metamorph inspect

    Flags:
      --all, -a                  Include external binaries (Temporal, Ollama, Chromium, Node)
      --ion-recipes              Inspect ion automation recipes
      --native                   Inspect native/bin/<platform>/ build output instead of AppData
      --show-backups             Show available backup versions (use with --ion-recipes)
      --show-pending             Show sites with pending status (use with --ion-recipes)

    Example:
        metamorph inspect
        metamorph inspect --all

  > ROLLOUT
    Deploy built binaries from repo to AppData

    Usage: metamorph rollout [--only <component>] [--dry-run]

    Arguments:
      [--only               required
      <component>]          required
      --dry-run             optional

    Flags:
      --dry-run                  Preview what would be copied without making changes
      --only                     Deploy a single component instead of all (brain, nucleus, sentinel, metamorph, host, workspace, setup, sensor, cortex, ionpump, vsix, bootstrap, hooks, config, nssm, ollama, temporal, node, runtime, chrome)

    Example:
        metamorph rollout
        metamorph rollout --dry-run
        metamorph rollout --only brain
        metamorph rollout --only ionpump
        metamorph rollout --only ionpump --dry-run
        metamorph --json rollout --only nucleus

    JSON Response:

======================================================[ IONPUMP ]=======================================================

  > ION-PUMP
    Ion site deployment and reconciliation

    Usage: metamorph ion-pump

    Subcommands:
      list                  List available ion reconciliation manifests
        metamorph ion-pump list [source]
      reconcile             Reconcile installed ions against a manifest
        metamorph ion-pump reconcile
      rollback              Rollback ion site to previous backup version
        metamorph ion-pump rollback <domain>
      status                Show installed ion site status
        metamorph ion-pump status [domain]
      verify                Verify SHA-256 integrity of installed ions
        metamorph ion-pump verify [domain]

    Subcommand Details:

      └─ LIST
        List available ion reconciliation manifests

        Usage: metamorph ion-pump list [source]

        Arguments:
          source                optional

        Flags:
          --source, -s               Filter by source: local | remote | all [default: all]

        Example:
            metamorph ion-pump list
            metamorph ion-pump list local
            metamorph ion-pump list --source remote
            metamorph --json ion-pump list

        JSON Response:
          {
            "manifests": [
              {
                "source": "local",
                "path": "installer/native/ionpump/bootstrap-ions.json",
                "ions_count": 1,
                "schema_version": "2.0"
              }
            ]
          }

        ----------------------------------------------------------------------

      └─ RECONCILE
        Reconcile installed ions against a manifest

        Usage: metamorph ion-pump reconcile

        Flags:
          --dry-run                  Simulate reconciliation without writing to ionsites/
          --force-swap               Bypass Brain signal — for bootstrap/testing without Nucleus
          --manifest, -m             Path to ion reconciliation manifest (required)

        Example:
            metamorph ion-pump reconcile --manifest installer/native/ionpump/bootstrap-ions.json --force-swap
            metamorph ion-pump reconcile --manifest /path/to/batcave-manifest.json
            metamorph ion-pump reconcile --manifest bootstrap-ions.json --force-swap --dry-run
            metamorph --json ion-pump reconcile --manifest bootstrap-ions.json --force-swap

        JSON Response:
          {
            "success": true,
            "manifest": "installer/native/ionpump/bootstrap-ions.json",
            "ions_processed": 1,
            "ions_swapped": 1,
            "ions_skipped": 0,
            "ions_failed": 0,
            "force_swap": true,

              {
                "domain": "github.com",

      └─ ROLLBACK
