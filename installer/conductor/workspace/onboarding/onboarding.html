Step 1 — GitHub
Creating a Personal
Access Token.
Bloom necesita un token clásico de GitHub con dos permisos:
repo y read:org.

Cómo crearlo:
1. Abrí GitHub → avatar → Settings
2. Bajá hasta Developer settings (último ítem)
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Marcá repo y read:org
6. Generá y copiá el token

⚠ Usá Tokens (classic), no Fine-grained.
El token empieza con ghp_ — la extensión lo detecta automáticamente al copiarlo.
