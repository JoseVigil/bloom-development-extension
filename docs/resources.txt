⚠️ Nota importante para Linux que no aplica a macOS: existen variantes gnu (glibc) y musl. La gnu es la correcta para distros estándar (Ubuntu, Debian, Fedora, etc.) — usá musl solo si vas a correr sobre Alpine. También hay variantes con sufijo de CPU microarquitectura (-v2, -v3, -v4 para AVX) pero la base sin sufijo es la más compatible y la recomendada para distribución general.
Misma estructura que macOS al descomprimir: carpeta python/bin/python3.10, python/lib/, etc. — mapea igual a installer/resources/runtime-linux/x86_64/.
